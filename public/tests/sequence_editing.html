Sequence Editing

To test the code sequence editing functionality, open the console and type the following commands
  • sequence.tracks.splice(3, 1) : the snare track should disappear
  • sequence.tracks.push(extraTracks[0]) : the tuba track should be added
  • When removing a track, the play line should shorten to be the correct length for the height of the sequence
  • When adding a track, the loading behavior should be the same as the normal track loading behavior
